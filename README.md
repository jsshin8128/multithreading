# Multithreading

Java 스레드의 기본인 `Runnable`부터 Java 21에서 정식 도입된 `Virtual Threads`까지, 코드를 작성하고 실행하며 학습하는 레포지토리입니다.

각 단계에서 발생하는 문제를 경험하고 해결해 나가면서, 해당 기술이 왜 필요한지 이해하는 것을 목표로 합니다.

## 학습 목표

* 하나의 JVM 프로세스 안에서 여러 스레드가 동작하는 구조를 이해하는 것이 목표입니다.
* 여러 스레드가 공유 자원에 접근할 때 발생하는 문제와 해결 방법을 학습하는 것이 목표입니다.
* 스레드 풀과 비동기 처리를 활용해 확장성 있는 코드를 작성하는 것이 목표입니다.
* 플랫폼 스레드의 한계를 이해하고, Java 21의 가상 스레드를 활용해 확장하는 것이 목표입니다.
